about pickr

pickr is an app that provides adequate analytics and data display for FIRST Robotics team scouting systems. pickr aims to simplify google sheets based scouting systems by offloading most of the data analysis and visualization so that teams can focus on scouting and making the best picks possible.


getting started

Currently, pickr is in closed alpha phase. A getting started guide for alpha testers will be posted soon.


tech

pickr was built using the Revel framework and Golang.


project "roadmap"

Here's a rough list of features to come for pickr:

  • Read data from The Blue Alliance via the tbago api
  • Read data from team provided scouting systems using google sheets api
  • Support for multiple user teams
  • Auto-picklist generation using team-provided or -configured weighting factors
  • Picklist management and exporting to google sheets
  • Live picklist management with auto-refresh for alliance selections
