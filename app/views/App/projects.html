Personal Site

Summer 2017 - Current

Created a personal site to showcase the various projects I've completed over the years. The previous iteration of this site was hosted statically on Github Pages and the Jekyll framework. This current iteration is written with the Revel framework in Go, deployed on Heroku with Docker. Content for this site is powered by an SQLite database for quick deployment.
I'm most proud of the Combat Robots section that I use to document my robots and fights. It reads from information stored in a SQLite database so that I can update information in one table vs modifying HTML across multiple pages.

The Maroon Alliance

Summer 2018 - Summer 2019

Score and ranking display for Gryffingear youth robotics events. Built using a Java application running on an Apache server. Pulls data from a Google sheet for analysis and display. Currently deployed on a Heroku free dyno.

Laser Cut Battery Caddy for Classrooms

Summer 2019

Custom designed and manufactured battery caddy for classroom STEM technology when an off-the-shelf solution did not exist. Designed in Autodesk Inventor and cut using a 130W laser cutter out of Acrylic. Designed during my stint as The Palmdale Aerospace Academy's Network Systems Admin.

3D Printed Yoyo

Spring 2018

Second attempt at a playable 3D printed yoyo. Designed using Autodesk Inventor and manufactured on a Stratasys uPrint 3d printer. Featured CNC machined aluminum weight rings to maximize moment of inertia.

frc-parts

Summer 2017

Platform for the archival and documentation of commonly-used and niche parts in FIRST Robotics Competition mechanical and electrical systems design. The foundation of this project is a spreadsheet containing information on parts, vendors, applications, etc. The most recent iteration of this project involved a Java application that would automatically generate markdown entries for all parts in the spreadsheet, to be hosted statically using Github Pages.

frc-oracle

Spring 2016

GUI for accessing analytics generated by the montecarlo simulator. Built using an incredibly hacky combination of windows batch scripts and github static pages to display data generated by a java app.

Montecarlo FRC Match Simulator

Spring 2015

Java-based simulator and prediction engine for FIRST Robotics Competition Matches. Analytics generated were discussed on GameSense S02E08. Results and community discussion can be found here. This was the first analysis of its kind to be done on FIRST robotics competition matches.

Genetic PID Tuner Platform

Summer 2015

An exploration of ideal PID gain tuning using genetic algorithms. Allowed user to specify input parameters and ideal conditions to find and simulate ideal PID gains over thousands of iterations. The Java application approximated real life physics to simulate system performance over time.

Thermal Shutoff Sensor Test Fixture

Spring 2015



Designed and oversaw testing for thermal sensors to be evaluated for use in SOFIA instrumentation. Using AutoCAD, I designed a test fixture to verify the thermal performance of Off the Shelf thermal shutoff sensors. Using plastic bushings, the fixture was optimized to produce minimal change in thermal performance of the sensors being tested. Coordinated with machine shop to procure materials for, manufacture, and assemble the test fixture. Developed detailed test plans to test the sensors under many thermal conditions and atmospheric pressures to verify sensor performance. Designed as part of my term at the NASA Armstrong Flight Research Center.

Robot Parts Database

2013 - Current

Compiled and maintained an aggregate list of parts commonly used in the FIRST Robotics Competition. Traditionally, these items and their applications would be scattered across the internet in various group chats and forum posts. This project aims to centralize this information for easy reference in the time crunch of the robot season.

Custom Arcade Button MIDI Controller

Summer 2015

Fully custom MIDI controller project utilizing an Arduino UNO. Utilized a digital input multiplexer to monitor the state of 12 arcade buttons to generate MIDI commands to send to a computer over USB. Designed, assembled, and tested in under 2 weeks. Operated in performance environments for over 3 years.

Video Game Controller MIDI Controller Mod

Summer 2013 - Summer 2014

Converting DJ Hero and Wii Nunchuk controllers to a fully functional MIDI controller for DJ performance software utilizing an Arduino MEGA. Features modular electrical design to accomodate a variety of novel sensors and input devices.

Modified MIDI Keyboard

Summer 2012 - Summer 2013

Modified second-hand MIDI Keyboard into a functional MIDI controller utilizing a Maple ARM microcontroller.
